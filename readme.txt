== Description ==
Connects WooCommerce to Machool to provide realtime shipping rates.
= Compatibility =
This plugin requires WooCommerce 8.4.0 or later.
= Ship locally and globally =
Set up a Machool account at [app.machool.com](https://app.machool.com) to connect your WooCommerce store to the Machool platform, generate a new API key and you're good to go!
